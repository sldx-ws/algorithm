
### 2. 确定一个数n的二进制表示中第x位（约定从0开始，方便）是0还是1

```
(n >> x) & 1
```

### 3. 将一个数n的二进制表示中第x位修改成1

```
n |= (1 << x)
```

### 4. 将一个数n的二进制表示中第x位修改成0

```
n &= (~(1 << x))
```

### 6. lowbit:  提取一个数n的二进制表示中最右侧的1

```
n & -n
// -n的补码就是n的最右边的1不动，该1的左边全部取反
```

### 7. 布莱恩・克尼汉算法: 将一个数n的二进制表示中最右侧的1修改成0

```
n & (n-1)
// n-1就是将最右侧的1及其右侧全部取反
//    n   : 101001000 100
//   n-1  : 101001000 011 
// n&(n-1): 101001000 000
```
