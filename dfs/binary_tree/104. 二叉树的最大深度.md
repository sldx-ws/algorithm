[104. 二叉树的最大深度](https://leetcode.cn/problems/maximum-depth-of-binary-tree/)

给定一个二叉树 `root` ，返回其最大深度。

二叉树的 **最大深度** 是指从根节点到最远叶子节点的最长路径上的节点数。

 

**示例 1：**

![img](https://assets.leetcode.com/uploads/2020/11/26/tmp-tree.jpg)

 

```
输入：root = [3,9,20,null,null,15,7]
输出：3
```

**示例 2：**

```
输入：root = [1,null,2]
输出：2
```

 

**提示：**

- 树中节点的数量在 `[0, 104]` 区间内。
- `-100 <= Node.val <= 100`



## 解法1：前序遍历

```c
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     struct TreeNode *left;
 *     struct TreeNode *right;
 * };
 */
#define max(a, b) ((a) > (b) ? (a) : (b))
int ret;

void prevOrder(struct TreeNode* root, int cur_height)
{
    if (!root) return;

    ret = max(ret, cur_height);

    prevOrder(root->left, cur_height + 1);
    prevOrder(root->right, cur_height + 1);
}
 
int maxDepth(struct TreeNode* root) {
    ret = 0;

    prevOrder(root, 1);

    return ret;
}
```



## 解法2：后序遍历

**根节点最大深度 = 根节点的最大高度**

```c
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     struct TreeNode *left;
 *     struct TreeNode *right;
 * };
 */
int postOrder(struct TreeNode* root)
{
    if (root == NULL) return 0;

    int left = postOrder(root->left);
    int right = postOrder(root->right);

    return 1 + (left > right ? left : right);
}

int maxDepth(struct TreeNode* root) {
    return postOrder(root);
}
```

## 补充：二叉树的深度和高度

```
深度：二叉树中任意节点到 根节点 的距离  // 前序遍历
高度：二叉树中任意节点到 叶节点 的距离  // 后序遍历
```
