[111. 二叉树的最小深度](https://leetcode.cn/problems/minimum-depth-of-binary-tree/)2

给定一个二叉树，找出其最小深度。

最小深度是从根节点到最近叶子节点的最短路径上的节点数量。

**说明：**叶子节点是指没有子节点的节点。

 

**示例 1：**

![img](https://assets.leetcode.com/uploads/2020/10/12/ex_depth.jpg)

```
输入：root = [3,9,20,null,null,15,7]
输出：2
```

**示例 2：**

```
输入：root = [2,null,3,null,4,null,5,null,6]
输出：5
```

 

**提示：**

- 树中节点数的范围在 `[0, 105]` 内
- `-1000 <= Node.val <= 1000`



## 解法1：前序遍历

```c
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     struct TreeNode *left;
 *     struct TreeNode *right;
 * };
 */
int min_depth;

void preOrder(struct TreeNode* root, int cur_depth)
{
    if (root == NULL) return;
    if (!root->left && !root->right)
    {
        if (cur_depth < min_depth)
            min_depth = cur_depth;

        return;
    }

    preOrder(root->left, cur_depth + 1);
    preOrder(root->right, cur_depth + 1);
}

int minDepth(struct TreeNode* root) {
    min_depth = INT_MAX;

    preOrder(root, 1);

    return min_depth == INT_MAX ? 0 : min_depth;
}
```

## 解法2：后序遍历

**根节点最小深度 = 根节点最小高度**

```c
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     struct TreeNode *left;
 *     struct TreeNode *right;
 * };
 */
int postOrder(struct TreeNode* root)
{
    if (root == NULL) return 0;

    int l = postOrder(root->left);
    int r = postOrder(root->right);

    if (root->left == NULL && root->right != NULL)
        return 1 + r;
    else if (root->left != NULL && root->right == NULL)
        return 1 + l;
    else 
        return 1 + (l < r ? l : r);
}

int minDepth(struct TreeNode* root) {
    return postOrder(root);
}
```

## 补充：二叉树的深度和高度

```
深度：二叉树中任意节点到 根节点 的距离  // 前序遍历
高度：二叉树中任意节点到 叶节点 的距离  // 后序遍历
```
