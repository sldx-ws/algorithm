[215. 数组中的第K个最大元素](https://leetcode.cn/problems/kth-largest-element-in-an-array/)

给定整数数组 `nums` 和整数 `k`，请返回数组中第 `**k**` 个最大的元素。

请注意，你需要找的是数组排序后的第 `k` 个最大的元素，而不是第 `k` 个不同的元素。

你必须设计并实现时间复杂度为 `O(n)` 的算法解决此问题。

 

**示例 1:**

```
输入: [3,2,1,5,6,4], k = 2
输出: 5
```

**示例 2:**

```
输入: [3,2,3,1,2,4,5,5,6], k = 4
输出: 4
```

 

**提示：**

- `1 <= k <= nums.length <= 105`
- `-104 <= nums[i] <= 104`



## 解法 - 快速选择 O(n)

- 1. 用“数组分三块”的思想（[75. 颜色分类](https://leetcode.cn/problems/sort-colors/)）
- 2. 随机的方式选取枢轴元素

- 注：若用堆排序，则O(nlogn)

```c
void my_swap(int* a, int* b)
{
    int tmp = *a;
    *a = *b;
    *b = tmp;
}

void partition(int* nums, int a, int b, int* outLeft, int* outRight)
{
    int r = rand();
    int pivot = nums[a + r % (b - a + 1)];
    int left = a - 1;
    int i = a;
    int right = b + 1;

    while (i < right)
    {
        if (nums[i] < pivot) 
            my_swap(&nums[++left], &nums[i++]); 
        else if (nums[i] == pivot) 
            ++i;
        else 
            my_swap(&nums[--right], &nums[i]);  // i不能++
    }   

    *outLeft = left;
    *outRight = right;
}

int quickSelect(int* nums, int left, int right, int x)
{
    if (left >= right) return nums[left];
    int outLeft = 0;
    int outRight = 0;

    partition(nums, left, right, &outLeft, &outRight);
    if (outLeft < x && x < outRight)
        return nums[x];
    else if (left <= x && x <= outLeft)
        return quickSelect(nums, left, outLeft, x);
    else 
        return quickSelect(nums, outRight, right, x);
}

int findKthLargest(int* nums, int numsSize, int k) {
    return quickSelect(nums, 0, numsSize - 1, numsSize - k);
}
```
