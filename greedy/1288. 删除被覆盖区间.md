[1288. 删除被覆盖区间](https://leetcode.cn/problems/remove-covered-intervals/)

给你一个区间列表，请你删除列表中被其他区间所覆盖的区间。

只有当 `c <= a` 且 `b <= d` 时，我们才认为区间 `[a,b)` 被区间 `[c,d)` 覆盖。

在完成所有删除操作后，请你返回列表中剩余区间的数目。

 

**示例：**

```
输入：intervals = [[1,4],[3,6],[2,8]]
输出：2
解释：区间 [3,6] 被区间 [2,8] 覆盖，所以它被删除了。
```

 

**提示：**

- `1 <= intervals.length <= 1000`
- `0 <= intervals[i][0] < intervals[i][1] <= 10^5`
- 对于所有的 `i != j`：`intervals[i] != intervals[j]`



## 解法：排序+贪心

- 时间复杂度: O(nlogn)
- 空间复杂度: O(1)

```c
int cmp(const void* pa, const void* pb)
{
    // 左端点相同时，右端点大的排前面
    int* a = *(int**)pa;
    int* b = *(int**)pb;
    return a[0] != b[0] ? a[0] - b[0] : b[1] - a[1];
}

int removeCoveredIntervals(int** intervals, int intervalsSize, int* intervalsColSize) {
    qsort(intervals, intervalsSize, sizeof(int*), cmp);
    int right_max = 0;  // 已遍历区间中的最大右端点
    int ret = 0;

    for (int i = 0; i < intervalsSize; ++i)
    {
        // 由区间左端点是从小到大排序，已遍历区间左端点 <= 当前区间左端点
        // 如果当前区间右端点 <= max_right，说明当前区间被另一个区间覆盖
        if (intervals[i][1] > right_max)
        {
            right_max = intervals[i][1];
            ++ret;
        }
    }

    return ret;
}
```
