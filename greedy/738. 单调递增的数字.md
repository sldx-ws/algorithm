[738. 单调递增的数字](https://leetcode.cn/problems/monotone-increasing-digits/)

当且仅当每个相邻位数上的数字 `x` 和 `y` 满足 `x <= y` 时，我们称这个整数是**单调递增**的。

给定一个整数 `n` ，返回 *小于或等于 `n` 的最大数字，且数字呈 **单调递增*** 。

 

**示例 1:**

```
输入: n = 10
输出: 9
```

**示例 2:**

```
输入: n = 1234
输出: 1234
```

**示例 3:**

```
输入: n = 332
输出: 299
```

 

**提示:**

- `0 <= n <= 10^9`



## 解法

```c
void my_itoa(int n, char* str, int* strSize)
{
    // 统计n的位数
    int tmp = n;
    int len = 0;
    while (tmp > 0)
    {
        tmp /= 10;
        len++;
    }

    str[len] = '\0';
    *strSize = len;

    while (len > 0)
    {
        str[--len] = n % 10 + '0';
        n /= 10;
    }
}

int monotoneIncreasingDigits(int n) {
    if (n < 10) return n;

    char buffer[11];
    int len = 0;
    my_itoa(n, buffer, &len);

    // 100...0 无法处理
    // for (int i = len - 1; i >= 1; --i)
    // {
    //     if (buffer[i - 1] > buffer[i])
    //     {
    //         buffer[i - 1]--;
    //         buffer[i] = '9';
    //     }
    // }

    int flag = len;  // 不能赋别的值
    for (int i = len - 1; i >= 1; --i)
    {
        if (buffer[i - 1] > buffer[i])
        {
            buffer[i - 1]--;
            buffer[i] = '9';
            flag = i;
        }
    }

    for (int i = flag; i < len; ++i)
    {
        buffer[i] = '9';
    }

    return atoi(buffer);
}
```
