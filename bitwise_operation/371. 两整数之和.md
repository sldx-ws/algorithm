[371. 两整数之和](https://leetcode.cn/problems/sum-of-two-integers/)

给你两个整数 `a` 和 `b` ，**不使用** 运算符 `+` 和 `-` ，计算并返回两整数之和。

 

**示例 1：**

```
输入：a = 1, b = 2
输出：3
```

**示例 2：**

```
输入：a = 2, b = 3
输出：5
```

 

**提示：**

- `-1000 <= a, b <= 1000`



## 解法

```c
int getSum(int a, int b) {
    while (b != 0)
    {
        int xor = a ^ b;
        unsigned int carry = (unsigned int)(a & b) << 1;
        a = xor;
        b = carry;
    }

    return a;
}
```
