[685. 冗余连接 II](https://leetcode.cn/problems/redundant-connection-ii/)

在本问题中，有根树指满足以下条件的 **有向** 图。该树只有一个根节点，所有其他节点都是该根节点的后继。该树除了根节点之外的每一个节点都有且只有一个父节点，而根节点没有父节点。

输入一个有向图，该图由一个有着 `n` 个节点（节点值不重复，从 `1` 到 `n`）的树及一条附加的有向边构成。附加的边包含在 `1` 到 `n` 中的两个不同顶点间，这条附加的边不属于树中已存在的边。

结果图是一个以边组成的二维数组 `edges` 。 每个元素是一对 `[ui, vi]`，用以表示 **有向** 图中连接顶点 `ui` 和顶点 `vi` 的边，其中 `ui` 是 `vi` 的一个父节点。

返回一条能删除的边，使得剩下的图是有 `n` 个节点的有根树。若有多个答案，返回最后出现在给定二维数组的答案。

 

**示例 1：**

![img](https://assets.leetcode.com/uploads/2020/12/20/graph1.jpg)

```
输入：edges = [[1,2],[1,3],[2,3]]
输出：[2,3]
```

**示例 2：**

![img](https://assets.leetcode.com/uploads/2020/12/20/graph2.jpg)

```
输入：edges = [[1,2],[2,3],[3,4],[4,1],[1,5]]
输出：[4,1]
```

 

**提示：**

- `n == edges.length`
- `3 <= n <= 1000`
- `edges[i].length == 2`
- `1 <= ui, vi <= n`



## 解法

```c
/**
 * Note: The returned array must be malloced, assume caller calls free().
 */
int father[1001];

int find(int x)
{
	int root = x;
	while (father[root] >= 0) root = father[root];
	
	// 路径压缩
	while (father[x] >= 0)
	{
		int parent = father[x];
		father[x] = root;
		
		x = parent;
	}
	
	return root;
}

void _union(int x, int y)
{
	int rootx = find(x);
	int rooty = find(y);
	
	if (rootx == rooty) return;
	
	// 控制数量小的往大的合并
	if (father[rootx] < father[rooty])
	{
		father[rootx] += father[rooty];
		father[rooty] = rootx;
	}
	else
	{
		father[rooty] += father[rootx];
		father[rootx] = rooty;
	}	
}

// 并查集不能判断有向图是否有环
// 这里删除一条边后，判断剩下的图是不是一棵树
bool isTreeAfterDeleteEdge(int** edges, int edgesSize, int rm)
{
    memset(father, -1, sizeof(father));  // 每次调用都要重置

    for (int i = 0; i < edgesSize; ++i)
    {
        if (i == rm) continue;

        if (find(edges[i][0]) == find(edges[i][1]))
            return false;
        else 
            _union(edges[i][0], edges[i][1]);
    }

    return true;
}

int* findRedundantDirectedConnection(int** edges, int edgesSize, int* edgesColSize, int* returnSize) {
    int in_degree[1001] = { 0 };
    *returnSize = 2;
    memset(father, -1, sizeof(father));

    // 1. 存在 入度=2 
    int vec[2] = { 0 };
    int count = 0;

    for (int i = 0; i < edgesSize; ++i)
    {
        ++in_degree[edges[i][1]];
    }

    // 从后往前，收集所有造成 入度=2 的边
    for (int i = edgesSize - 1; i >= 0; --i)
    {
        if (in_degree[edges[i][1]] == 2)
        {
            vec[count++] = i; // 存两条边的下标
        }
    }

    if (count > 0)
    {
        if (isTreeAfterDeleteEdge(edges, edgesSize, vec[0]))  // 先判断后出现的边vec[0]
            return edges[vec[0]];
        else 
            return edges[vec[1]];
    }

    // 2. 没有 入度=2，但存在 有向环(首尾相接，顶点数=边数)
    memset(father, -1, sizeof(father));  // 重置
    for (int i = 0; i < edgesSize; ++i)
    {
        if (find(edges[i][0]) == find(edges[i][1]))
            return edges[i];
        else 
            _union(edges[i][0], edges[i][1]);
    }

    return NULL;
}
```

###### 题目描述

有一种有向树,该树只有一个根节点，所有其他节点都是该根节点的后继。该树除了根节点之外的每一个节点都有且只有一个父节点，而根节点没有父节点。有向树拥有 n 个节点和 n - 1 条边。如图： 

![img](https://file.kamacoder.com/pics/20250326162639.png)



现在有一个有向图，有向图是在有向树中的两个没有直接链接的节点中间添加一条有向边。如图：

![img](https://file.kamacoder.com/pics/20250326162836.png)





输入一个有向图，该图由一个有着 n 个节点(节点编号 从 1 到 n)，n 条边，请返回一条可以删除的边，使得删除该条边之后该有向图可以被当作一颗有向树。

###### 输入描述

第一行输入一个整数 N，表示有向图中节点和边的个数。 

后续 N 行，每行输入两个整数 s 和 t，代表这是 s 节点连接并指向 t 节点的单向边

###### 输出描述

输出一条可以删除的边，若有多条边可以删除，请输出标准输入中最后出现的一条边。

###### 输入示例

```
3
1 2
1 3
2 3
```

###### 输出示例

```
2 3
```

![img](https://kamacoder.com/upload/kamacoder.com/image/20240418/20240418110555_68889.png)
