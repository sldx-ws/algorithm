# P3366 【模板】最小生成树

## 题目描述

如题，给出一个无向图，求出最小生成树，如果该图不连通，则输出 `orz`。

## 输入格式

第一行包含两个整数 $N,M$，表示该图共有 $N$ 个结点和 $M$ 条无向边。

接下来 $M$ 行每行包含三个整数 $X_i,Y_i,Z_i$，表示有一条长度为 $Z_i$ 的无向边连接结点 $X_i,Y_i$。

## 输出格式

如果该图连通，则输出一个整数表示最小生成树的各边的长度之和。如果该图不连通则输出 `orz`。

## 输入输出样例 #1

### 输入 #1

```
4 5
1 2 2
1 3 2
1 4 3
2 3 4
3 4 3
```

### 输出 #1

```
7
```

## 说明/提示

数据规模：

对于 $20\%$ 的数据，$N\le 5$，$M\le 20$。

对于 $40\%$ 的数据，$N\le 50$，$M\le 2500$。

对于 $70\%$ 的数据，$N\le 500$，$M\le 10^4$。

对于 $100\%$ 的数据：$1\le N\le 5000$，$1\le M\le 2\times 10^5$，$1\le Z_i \le 10^4$，$1\le X_i,Y_i\le N$。

所以最小生成树的总边权为 $2+2+3=7$。



## 解法

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <limits.h>  // INT_MAX
#include <stdbool.h> // bool

#define MAX 50001

int primMST(int** grid, int n)
{
    int totalWeight = 0;  // MST总权值
    
    int parent[MAX];      // 保存MST
    int lowCost[MAX];     // 最小权值
    bool visited[MAX];    // 是否加入生成树
    
    memset(parent, -1, sizeof(int) * (n + 1));
    memset(visited, 0, sizeof(int) * (n + 1));
    for (int i = 0; i < n + 1; ++i) lowCost[i] = INT_MAX;

    // 从 1 号顶点开始
    parent[1] = -1;
    lowCost[1] = 0;

    // 执行 v-1 次 （v-1条边）
    for (int count = 1; count <= n - 1; ++count)
    {
        // 找到未访问的、lowCost 最小的顶点 u
        int min = INT_MAX;
        int u = -1;

        for (int i = 1; i < n + 1; ++i)
        {
            if (!visited[i] && lowCost[i] < min)
            {
                min = lowCost[i];
                u = i;
            }
        }

        if (u == -1) return -1;  // 图不连通
        else visited[u] = true;

        // 更新相邻节点
        for (int i = 1; i < n + 1; ++i)
        {
            if (!visited[i] && grid[u][i] != 0 && grid[u][i] < lowCost[i])
            {
                parent[i] = u;
                lowCost[i] = grid[u][i];
            }
        }
    }

    for (int i = 2; i < n + 1; ++i)  // 必须从2号节点开始
        totalWeight += grid[i][parent[i]];

    return totalWeight;
}

int main()
{
    int n, m;
    scanf("%d %d", &n, &m);
    
    int** grid = (int**)malloc(sizeof(int*) * (n + 1));
    for (int i = 0; i < n + 1; ++i)
    {
        grid[i] = (int*)malloc(sizeof(int) * (n + 1));
        memset(grid[i], 0, sizeof(int) * (n + 1));
    }

    while (m--)
    {
        int x, y, z;
        scanf("%d %d %d", &x, &y, &z);
        
        // 处理重边
        if (grid[x][y] == 0 || z < grid[x][y])
        {
            grid[x][y] = z;
            grid[y][x] = z;
        }
    }

    int ret = primMST(grid, n);
    if (ret > 0)
        printf("%d", ret);
    else 
        printf("orz");

    return 0;
}
```
