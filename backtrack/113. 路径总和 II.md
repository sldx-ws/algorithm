[113. 路径总和 II](https://leetcode.cn/problems/path-sum-ii/)

给你二叉树的根节点 `root` 和一个整数目标和 `targetSum` ，找出所有 **从根节点到叶子节点** 路径总和等于给定目标和的路径。

**叶子节点** 是指没有子节点的节点。

 

**示例 1：**

![img](https://assets.leetcode.com/uploads/2021/01/18/pathsumii1.jpg)

```
输入：root = [5,4,8,11,null,13,4,7,2,null,null,5,1], targetSum = 22
输出：[[5,4,11,2],[5,8,4,5]]
```

**示例 2：**

![img](https://assets.leetcode.com/uploads/2021/01/18/pathsum2.jpg)

```
输入：root = [1,2,3], targetSum = 5
输出：[]
```

**示例 3：**

```
输入：root = [1,2], targetSum = 0
输出：[]
```

 

**提示：**

- 树中节点总数在范围 `[0, 5000]` 内
- `-1000 <= Node.val <= 1000`
- `-1000 <= targetSum <= 1000`



## 解法

```c
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     struct TreeNode *left;
 *     struct TreeNode *right;
 * };
 */
/**
 * Return an array of arrays of size *returnSize.
 * The sizes of the arrays are returned as *returnColumnSizes array.
 * Note: Both returned array and *columnSizes array must be malloced, assume caller calls free().
 */
int** ret;
int ret_len;
 
int* path;
int path_len;

void dfs(struct TreeNode* root, int targetSum, int* returnColumnSizes)
{
    if (!root) return;

    targetSum -= root->val;
    path[path_len++] = root->val;
    if (!root->left && !root->right)
    {
        if (targetSum == 0)
        {
            ret[ret_len] = (int*)malloc(sizeof(int) * path_len);
            for (int i = 0; i < path_len; ++i)
                ret[ret_len][i] = path[i];

            returnColumnSizes[ret_len] = path_len;
            ++ret_len;
        }

        // 回溯
        targetSum += root->val;
        --path_len;

        return;
    }

    dfs(root->left, targetSum, returnColumnSizes);
    dfs(root->right, targetSum, returnColumnSizes);
    // 回溯
    targetSum += root->val;
    --path_len;
}

int** pathSum(struct TreeNode* root, int targetSum, int* returnSize, int** returnColumnSizes) {
    ret = (int**)malloc(sizeof(int*) * 32768);
    path = (int*)malloc(sizeof(int) * 5000);
    ret_len = path_len = 0;
    *returnColumnSizes = (int*)malloc(sizeof(int) * 32768);
 
    dfs(root, targetSum, *returnColumnSizes);

    *returnSize = ret_len;
    return ret;
}
```
