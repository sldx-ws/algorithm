[784. 字母大小写全排列](https://leetcode.cn/problems/letter-case-permutation/)

给定一个字符串 `s` ，通过将字符串 `s` 中的每个字母转变大小写，我们可以获得一个新的字符串。

返回 *所有可能得到的字符串集合* 。以 **任意顺序** 返回输出。

 

**示例 1：**

```
输入：s = "a1b2"
输出：["a1b2", "a1B2", "A1b2", "A1B2"]
```

**示例 2:**

```
输入: s = "3z4"
输出: ["3z4","3Z4"]
```

 

**提示:**

- `1 <= s.length <= 12`
- `s` 由小写英文字母、大写英文字母和数字组成



## 解法

```c
/**
 * Note: The returned array must be malloced, assume caller calls free().
 */
char** ret;
int ret_len;

char* path;
int path_len;

void dfs(char* s, int len, int start)
{
    if (start == len)
    {
        path[path_len] = '\0';
        ret[ret_len] = (char*)malloc(sizeof(char) * (path_len + 1));
        strcpy(ret[ret_len], path);
        ++ret_len;

        return;
    }

    char ch = s[start];
    // 不改变
    path[path_len++] = ch;
    dfs(s, len, start + 1);
    --path_len;

    // 改变
    if (ch < '0' || ch > '9')
    {
        if (ch >= 'a' && ch <= 'z')
        {
            path[path_len++] = ch - 32;
            dfs(s, len, start + 1);
            --path_len;

        }
        else
        {
            path[path_len++] = ch + 32;
            dfs(s, len, start + 1);
            --path_len;
        }
    }
}

char** letterCasePermutation(char* s, int* returnSize) {
    int len = strlen(s);
    
    path = (char*)malloc(sizeof(char) * 13);
    ret = (char**)malloc(sizeof(char*) * 10001);
    path_len = ret_len = 0;
    
    dfs(s, len, 0);

    *returnSize = ret_len;

    return ret;
}
```
