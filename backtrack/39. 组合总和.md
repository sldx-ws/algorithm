[39. 组合总和](https://leetcode.cn/problems/combination-sum/)

给你一个 **无重复元素** 的整数数组 `candidates` 和一个目标整数 `target` ，找出 `candidates` 中可以使数字和为目标数 `target` 的 所有 **不同组合** ，并以列表形式返回。你可以按 **任意顺序** 返回这些组合。

`candidates` 中的 **同一个** 数字可以 **无限制重复被选取** 。如果至少一个数字的被选数量不同，则两种组合是不同的。 

对于给定的输入，保证和为 `target` 的不同组合数少于 `150` 个。

 

**示例 1：**

```
输入：candidates = [2,3,6,7], target = 7
输出：[[2,2,3],[7]]
解释：
2 和 3 可以形成一组候选，2 + 2 + 3 = 7 。注意 2 可以使用多次。
7 也是一个候选， 7 = 7 。
仅有这两种组合。
```

**示例 2：**

```
输入: candidates = [2,3,5], target = 8
输出: [[2,2,2,2],[2,3,3],[3,5]]
```

**示例 3：**

```
输入: candidates = [2], target = 1
输出: []
```

 

**提示：**

- `1 <= candidates.length <= 30`
- `2 <= candidates[i] <= 40`
- `candidates` 的所有元素 **互不相同**
- `1 <= target <= 40`



## 解法

```c
/**
 * Return an array of arrays of size *returnSize.
 * The sizes of the arrays are returned as *returnColumnSizes array.
 * Note: Both returned array and *columnSizes array must be malloced, assume caller calls free().
 */
int** ret;
int ret_len;

int* tmp;
int tmp_len;

int cmp(const void* a, const void* b)
{
    return *(int*)a - *(int*)b;
}

void backtracking(int* candidates, int candidatesSize, int cur, int target, int sum, int** returnColumnSizes)
{
    if (sum == target)
    {
        ret[ret_len] = (int*)malloc(sizeof(int) * tmp_len);
        for (int i = 0; i < tmp_len; ++i)
        {
            ret[ret_len][i] = tmp[i];
        }

        (*returnColumnSizes)[ret_len] = tmp_len;
        ++ret_len;

        return;
    }

    if (sum > target) return;

    for (int i = cur; i < candidatesSize; ++i)  // sort可以剪枝
    {
        tmp[tmp_len++] = candidates[i];
        sum += candidates[i];

        backtracking(candidates,candidatesSize, i, target, sum, returnColumnSizes);

        sum -= candidates[i];
        --tmp_len;
    }
}

int** combinationSum(int* candidates, int candidatesSize, int target, int* returnSize, int** returnColumnSizes) {
    qsort(candidates, candidatesSize, sizeof(int), cmp);  // 剪枝优化
    
    ret = (int**)malloc(sizeof(int*) * 150);
    tmp = (int*)malloc(sizeof(int) * 1000);
    *returnColumnSizes = (int*)malloc(sizeof(int) * 150);
    ret_len = tmp_len = 0;

    backtracking(candidates, candidatesSize, 0, target, 0, returnColumnSizes);

    *returnSize = ret_len;

    return ret;
}
```
