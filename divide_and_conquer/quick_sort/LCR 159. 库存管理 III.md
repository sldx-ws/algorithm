[LCR 159. 库存管理 III](https://leetcode.cn/problems/zui-xiao-de-kge-shu-lcof/)

仓库管理员以数组 `stock` 形式记录商品库存表，其中 `stock[i]` 表示对应商品库存余量。请返回库存余量最少的 `cnt` 个商品余量，返回 **顺序不限**。

 

**示例 1：**

```
输入：stock = [2,5,7,4], cnt = 1
输出：[2]
```

**示例 2：**

```
输入：stock = [0,2,3,6], cnt = 2
输出：[0,2] 或 [2,0]
```

 

**提示：**

- `0 <= cnt <= stock.length <= 10000`
- `0 <= stock[i] <= 10000`



## 解法 - 快速选择 O(n)

- 1. 用“数组分三块”的思想（[75. 颜色分类](https://leetcode.cn/problems/sort-colors/)）
- 2. 随机的方式选取枢轴元素

```c
/**
 * Note: The returned array must be malloced, assume caller calls free().
 */
void my_swap(int* a, int* b)
{
    int tmp = *a;
    *a = *b;
    *b = tmp;
}

void partition(int* nums, int a, int b, int* outLeft, int* outRight)
{
    int pivot = nums[a + rand() % (b - a + 1)];
    int left = a - 1; 
    int i  = a;
    int right = b + 1;

    while (i < right)
    {
        if (nums[i] < pivot)
            my_swap(&nums[++left], &nums[i++]);
        else if (nums[i] == pivot)
            ++i;
        else 
            my_swap(&nums[--right], &nums[i]);
    }

    *outLeft = left;
    *outRight = right;
}

void quickSelect(int* nums, int left, int right, int k)
{
    if (left >= right) return;

    int outLeft = 0;
    int outRight = 0;
    partition(nums, left, right, &outLeft, &outRight);
    if (left <= k && k <= outLeft)
        quickSelect(nums, left, outLeft, k);
    else if (outRight <= k && k <= right)
        quickSelect(nums, outRight, right, k);
    else 
        return;
}
 
int* inventoryManagement(int* stock, int stockSize, int cnt, int* returnSize) {
    quickSelect(stock, 0, stockSize - 1, cnt - 1);

    *returnSize = cnt;
    return stock;
}
```
