[343. 整数拆分](https://leetcode.cn/problems/integer-break/)

给定一个正整数 `n` ，将其拆分为 `k` 个 **正整数** 的和（ `k >= 2` ），并使这些整数的乘积最大化。

返回 *你可以获得的最大乘积* 。

 

**示例 1:**

```
输入: n = 2
输出: 1
解释: 2 = 1 + 1, 1 × 1 = 1。
```

**示例 2:**

```
输入: n = 10
输出: 36
解释: 10 = 3 + 3 + 4, 3 × 3 × 4 = 36。
```

 

**提示:**

- `2 <= n <= 58`



## 解法

```c
int my_max(int a, int b)
{
    return a > b ? a : b;
}

int integerBreak(int n) {
    // dp[i]:把i拆分成至少两个正整数
    int dp[n + 1];

    // dp[0] = 0;
    // dp[1] = 0;
    memset(dp, 0, sizeof(dp));

    for (int i = 2; i <= n; ++i)
    {
        // for (int j = 1; j < i; ++j)
        for (int j = 1; j <= i/2; ++j)
        {
            dp[i] = my_max(j * (i - j), my_max(j * dp[i - j], dp[i]));
        }
    }

    return dp[n];
}
```
