## 描述

你有一个背包，最多能容纳的体积是V。

现在有n种物品，每种物品有任意多个，第i种物品的体积为v~i~ ,价值为w~i~。

（1）求这个背包至多能装多大价值的物品？

（2）若背包恰好装满，求至多能装多大价值的物品？

### 输入描述：

第一行两个整数n和V，表示物品个数和背包体积。

接下来n行，每行两个数v~i~和w~i~，表示第i种物品的体积和价值。

1≤n,V≤10001≤*n*,*V*≤1000

### 输出描述：

输出有两行，第一行输出第一问的答案，第二行输出第二问的答案，如果无解请输出0。

## 示例1

输入：

```
2 6
5 10
3 1
```

输出：

```
10
2
```

## 示例2

输入：

```
3 8
3 10
9 1
10 1
```

输出：

```
20
0
```

说明：

```
无法恰好装满背包。
```

## 示例3

输入：

```
6 13
13 189
17 360
19 870
14 184
6 298
16 242
```

输出：

```
596
189
```

说明：

```
可以装5号物品2个，达到最大价值298*2=596，若要求恰好装满，只能装1个1号物品，价值为189.
```



## 状态转移方程推导

### 背包不用恰好装满

- `dp[i][j]`
  - 不选`i`：`dp[i - 1][j]`
  - 选`1`个`i`：`dp[i - 1][j - v[i]] + w[i]`
  - 选`2`个`i`：`dp[i - 1][j - 2* v[i]] + 2 * w[i]`
  - 选`3`个`i`：`dp[i - 1][j - 3 * v[i]] + 3 * w[i]`
  - `……`

- （1）因此：`dp[i][j] = max(dp\[i-1][j], `$\underline{dp[i-1][j-v[i]] + w[i], dp[i-1][j-2*v[i]] + 2*w[i], ..., dp[i-1][k*v[i]] + k*w[i]}$`)`
- （2）令`j = j-v[i]`：`dp[i][j-v[i]] = max(`$\underline{dp[i-1][j - v[i]], dp[i-1][j-2*v[i]] + w[i], dp[i-1][j-3*v[i]] + 2*w[i], ..., dp[i-1][x*v[i]] + (x-1)*w[i]}$`)`
- 显然：`(1)和(2)中 k = x`
- 综上：`(2)+w[i]后 等于 (1)中的下划线`
- 即：`dp[i][j] = max(dp[i-1][j], dp[i][j-v[i]] + w[i])`

### 背包恰好装满

- 同上：`dp[i][j] = max(dp[i-1][j], dp[i][j-v[i]] + w[i])`
  - 用`-1`表示没有这个状态
  - 即：`dp[i][j-v[i]] != -1`

## 解法

```c
#include <stdio.h>
#include <string.h>  // for memset

const int N = 1010;
int dp[N][N];

int n; // 物品个数
int v[N]; // 物品体积
int w[N]; // 物品价值
int bagWeight; // 背包容量

int max(int a, int b)
{
    return a > b ? a : b;
}

int main() 
{
    scanf("%d %d", &n, &bagWeight);
    for (int i = 1; i <= n; ++i) 
        scanf("%d %d", &v[i], &w[i]);

    // (1)
    // dp[i][j]: 从前i个物品中挑选，总体积"不超过"j的所有选法中最大价值
    // 物品与背包遍历顺序 可交换
    memset(dp, 0, sizeof(dp));
    for (int i = 1; i <= n; ++i)
    {
        for (int j = 0; j <= bagWeight; ++j)
        {
            // 不选i号物品
            dp[i][j] = dp[i-1][j];
            
            // 选n个i号物品
            if (j - v[i] >= 0)
                dp[i][j] = max(dp[i][j], dp[i][j-v[i]] + w[i]);
        }
    }

    printf("%d\n", dp[n][bagWeight]);


    // (2)
    // dp[i][j]: 从前i个物品中挑选，总体积"正好等于"j的所有选法中最大价值
    // dp[i][j]==-1 表示没有这种情况
    // 物品与背包遍历顺序 可交换
    memset(dp, 0, sizeof(dp));
    for (int j = 1; j <= bagWeight; ++j) dp[0][j] = -1;

    for (int i = 1; i <= n; ++i)
    {
        for (int j = 0; j <= bagWeight; ++j)
        {
            // 不选i号物品
            dp[i][j] = dp[i-1][j];
            
            // 选n个i号物品
            if (j - v[i] >= 0 && dp[i][j-v[i]] != -1)
                dp[i][j] = max(dp[i][j], dp[i][j-v[i]] + w[i]);
        }
    }

    printf("%d\n", dp[n][bagWeight] == -1 ? 0 : dp[n][bagWeight]);
    
    return 0;
}
```

### 空间优化：滚动数组

**注意：遍历顺序必须从右向左**

```c
#include <stdio.h>
#include <string.h>  // for memset

const int N = 1010;
int dp[N];

int n; // 物品个数
int v[N]; // 物品体积
int w[N]; // 物品价值
int bagWeight; // 背包容量

int max(int a, int b)
{
    return a > b ? a : b;
}

int main() 
{
    scanf("%d %d", &n, &bagWeight);
    for (int i = 1; i <= n; ++i) 
        scanf("%d %d", &v[i], &w[i]);

    // (1)
    // 物品与背包遍历顺序 可交换
    memset(dp, 0, sizeof(dp));
    for (int i = 1; i <= n; ++i)
    {
        // for (int j = 0; j <= bagWeight; ++j) // 从左向右（与01背包不同）
        for (int j = v[i]; j <= bagWeight; ++j) // 从左向右（与01背包不同）
        {
        // if (j - v[i] >= 0)
            dp[j] = max(dp[j], dp[j-v[i]] + w[i]);
        }
    }

    printf("%d\n", dp[bagWeight]);


    // (2) - 写法1
    // 物品与背包遍历顺序 可交换
    memset(dp, 0, sizeof(dp));
    for (int j = 1; j <= bagWeight; ++j) dp[j] = -1;

    for (int i = 1; i <= n; ++i)
    {
        // for (int j = 1; j <= bagWeight; ++j) // 从左向右（与01背包不同）
        for (int j = v[i]; j <= bagWeight; ++j) // 从左向右（与01背包不同）
        {
            // if (j - v[i] >= 0 && dpf[j-v[i]] != -1)
            if (dp[j-v[i]] != -1)
                dp[j] = max(dp[j], dp[j-v[i]] + w[i]);
        }
    }

    printf("%d\n", dp[bagWeight] == -1 ? 0 : dp[bagWeight]);
    
    return 0;
}
```

## 完全背包-补充

- 纯完全背包：问`装满背包的最大价值`或者`背包最多能装多大价值``
  - ``先遍历物品后遍历背包`或者`先遍历背包后遍历物品`是一样的

- 问`装满背包有多少种方法`，则需要区分求的是`组合数`还是`排列数`
  - 不强调集合里元素的顺序，则`先遍历物品再遍历背包` - `组合数`
  - 强调集合里元素的顺序，则`先遍历背包再遍历物品` - `排列数`
